## Meanings of words and stuff

Basho: tournament
Torikumi: Schadule (list of scheduled matches for a basho)
Kimarite: Winning move
Rikishi: Sumo competitor
Banzuke: Rank
Shikona: Rikishi's ring name
Yusho: Title for basho winner
